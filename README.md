# WordChat
KI-Chat Interface for Wordpress

#1.st command
create a wordpress plugin
zweck
- erstelle einen Chatbot
  - ollama (local an remote server)
  - claude
  - chatgpt

 Füge den Chatbot mit folgendem Shortcode in eine Seite ein:

[handschelle-chat]
Alle verfügbaren Attribute:

Attribut	Standard	Beschreibung
model	Standard-Modell (s.o.)	Ollama-Modellname
title	KI-Assistent	Titel im Chat-Header
system	Standard-Prompt (s.o.)	Systemanweisung an die KI
placeholder	Schreibe eine Nachricht …	Platzhalter im Eingabefeld
